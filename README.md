## 🎯 3 Ways to Use Your AI Assistant

### 1. 🌐 **Web Application** (Enhanced)
```bash
# Frontend (VS Code-like interface)
cd frontend && npm install && npm start

# Backend (Free models + enhanced context)
cd backend && pip install -r requirements.txt && python main.py
```
**Access**: http://localhost:3000

### 2. 🔌 **VS Code Extension** (New!)
```bash
cd vscode-extension && npm install && npm run package
# Install the .vsix file in VS Code
```
**Features**: Native chat panel, code commands, keyboard shortcuts

### 3. 💻 **Desktop Application** (New!)
```bash
cd desktop-app && npm install && npm run dev
# Builds for Windows, macOS, Linux
```
**Features**: Native menus, auto-updates, file system access

## 📚 Documentation

### 📖 **Complete Setup Guide**
See [INSTALLATION_GUIDE.md](INSTALLATION_GUIDE.md) for detailed instructions for all platforms.

### 🔧 **Configuration**
- **Web App**: Set API key in `backend/.env`
- **VS Code**: Settings → AI Assistant → OpenRouter API Key
- **Desktop**: File → Preferences → API Key

### 🐛 **Troubleshooting**
- **API Issues**: Verify OpenRouter API key
- **Context Problems**: Ensure files are open in editor
- **Model Issues**: Try different free models
